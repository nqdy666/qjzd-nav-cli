# CLI 参考

QJZD Nav CLI 命令行工具完整命令参考。

## 命令概览

| 命令                       | 说明                |
| -------------------------- | ------------------- |
| [auth](./auth)             | 认证和 Profile 管理 |
| [link](./link)             | 链接增删改查        |
| [category](./category)     | 分类增删改查        |
| [tag](./tag)               | 标签增删改查        |
| [backup](./backup)         | 备份和恢复          |
| [settings](./settings)     | 站点设置            |
| [completion](./completion) | Shell 补全脚本      |

## 通用选项

| 选项               | 说明               |
| ------------------ | ------------------ |
| `--profile <name>` | 指定使用的 Profile |
| `--json`           | 输出 JSON 格式     |
| `--help`, `-h`     | 显示帮助信息       |
| `--version`, `-v`  | 显示版本号         |
